# 用户说明书配图目录

本目录存放 **《用户操作说明书》（`docs/USER-MANUAL.md`）** 中引用的截图。

## 拍摄建议

- **分辨率**：浏览器窗口宽度建议 **1280px～1440px**（或全屏后截取主要内容区），格式 **PNG**。
- **隐私**：打码邮箱、订单号等敏感信息；可用测试账号截图。
- **命名**：与下表文件名**完全一致**（区分大小写），放入本目录后，Markdown 中的图片即可正常显示。
