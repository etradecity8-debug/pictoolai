## 一、使用前准备

### 1.1 运行环境

- 推荐使用 **Chrome** 或 **Edge** 等 Chromium 内核浏览器，并保持较新版本。
- 需要 **注册并登录** 后，方可使用生图、仓库、AI 运营助手等核心功能。
- 部分能力（如另存为到本地）依赖浏览器是否支持「文件选择器」；若批量保存文件夹被拒绝，可换成本地「文稿」下新建空文件夹再试。

### 1.2 术语速览

| 术语 | 含义 |
|------|------|
| 积分 | 使用会扣积分的功能时消耗；新用户注册通常有体验积分（以站内说明为准）。 |
| 仓库 | 账号下保存的生成图记录，按日期分组；可批量下载、删除或带图进入 AI 美工。 |
| 输出设置 | 多数生图页可选：模型、画面比例、清晰度等，并显示预估积分。 |

## 二、网站导航与账号

### 2.1 顶栏导航（登录后）

登录成功后，顶栏通常包含以下内容（**具体以您所见为准**；部分扩展入口可能未对全部账号开放）：

- **通用电商生图**：全品类产品组图流程。
- **AI美工**：单图编辑、修图、风格、水印等。
- **电商AI运营助手**：多平台 Listing 文案与配图相关能力。
- **浏览器扩展**：登录后在顶栏下载 **Chromium 扩展 zip**，本地解压后用「加载已解压的扩展程序」安装（详见第八节）。
- **订阅**：查看套餐与积分说明。
- **联系我们**：微信等联系方式。
- **工作台**：进入个人工作台（仓库、Listing 历史等）。

![图：顶栏导航总览](images/user-manual/01-nav-overview.png)

### 2.2 注册

1. 在首页或顶栏进入 **免费注册**。
2. 按页面提示填写邮箱、密码等信息并完成注册。
3. 若页面提示「新用户赠送体验积分」，请以实际显示为准。

![图：注册页](images/user-manual/02-register.png)

### 2.3 登录与退出

1. 点击 **登录**，输入注册邮箱与密码。
2. 登录成功后，顶栏会显示账号入口与 **工作台**。
3. 需要退出时，点击 **退出**。

![图：登录页](images/user-manual/03-login.png)

### 2.4 找回密码

在登录页选择 **忘记密码**（若已开放），按邮件或页面指引重置密码。

## 三、通用电商生图（全流程）

**路径**：顶栏 **通用电商生图**。

流程分为 **5 个步骤**：输入 → 分析中 → 确认规划 → 生成中 → 完成。

![图：步骤 1 输入区](images/user-manual/10-detailset-step1.png)

### 3.1 步骤 1：输入

1. **上传产品图**  
   - 点击上传区域，选择清晰的产品实拍图（建议主体完整、光线均匀）。

2. **填写产品名称**（必填）  
   - 用于分析与生成时的产品指代，请与真实 SKU 一致。

3. **选填**：目标人群、风格、其他要求  
   - 有助于统一画面气质与卖点表达。

4. **选择各类型图片张数**  
   - **白底主图**、**卖点图**、**特写图**、**场景图**、**交互图** 可分别选择数量（各有上限，以页面为准）。  
   - **白底主图** 无需逐条描述。  
   - **卖点图 / 场景图 / 特写图 / 交互图**：选择 N 张时，一般需要填写 **N 条对应描述**（页面会校验）。

5. **是否显示文字**  
   - 按类型勾选后，规划与生成会区分「图上是否出现文案」，请与上架平台习惯一致。

6. 确认 **目标语言**（若与画面文字相关）、后续会用到的 **模型 / 比例 / 清晰度**（部分在后续步骤或生成前也可调整，以页面为准）。

7. 点击 **开始分析**（或同等文案按钮）。  
   - 若提示未填必填项或总张数为 0，请按提示补全。

### 3.2 步骤 2：分析中

- 等待服务端完成图片与文字分析，请勿关闭页面。  
- 若网络不稳定，前端可能对分析请求自动重试（以实际行为为准）。

### 3.3 步骤 3：确认规划

1. 阅读 **整体设计规范** 与 **图片规划**（多为 Markdown 排版，含按类型的彩色标签）。  
2. 若需调整需求，可返回步骤 1 修改条件后重新分析（以页面「上一步」或「重新」类按钮为准）。  
3. 确认无误后，进入 **生成**。

![图：步骤 3 确认规划](images/user-manual/11-detailset-step3.png)

### 3.4 步骤 4：生成中

- 按规划逐张生成，请耐心等待。  
- 生成过程会消耗积分；失败一般不扣费（以站内规则为准）。

### 3.5 步骤 5：完成

1. 结果按 **图片类型** 分组展示，可点击放大预览。  
2. **保存到本地**：单张可另存为；多张可能引导选择文件夹（受浏览器限制，部分系统目录会失败）。  
3. 成功生成的图片通常会 **自动进入仓库**，无需单独「保存到仓库」。  
4. 若修改了产品信息、张数、描述等核心条件，一般需要 **回到步骤 1** 并清空后续结果（以页面提示为准）；仅改模型、语言、尺寸等，可能支持直接再次生成（以页面为准）。

![图：步骤 5 完成](images/user-manual/12-detailset-step5.png)

### 3.6 浏览器扩展联动（可选）

若已安装 PicToolAI 浏览器扩展，可在电商网页 **右键图片** 进入组图流程，并可能自动带上产品图、预选图片类型。详见 [第八节](#八浏览器扩展可选)。

## 四、AI 美工

**路径**：顶栏 **AI美工**，在左侧选择具体功能。

左侧为 **功能分类菜单**，右侧为具体操作区。多数流程为：**准备图片 → 填写参数 / 遮罩 → 输出设置 → 生成 → 保存 / 自动入仓库**。

![图：AI 美工侧栏](images/user-manual/20-aidesigner-sidebar.png)

### 4.1 通用操作

1. **图片来源**  
   - **本地上传**：按页面提示选择图片。  
   - **从作品库选择**：在支持该能力的页面，打开选图弹窗，从仓库选取（需登录）。

2. **输出设置**（页面下方或侧栏，视具体工具而定）  
   - 选择 **生图模型**、**画面比例**、**清晰度** 等。  
   - 查看 **预估积分**；实际扣费以生成成功为准。

![图：输出设置示例](images/user-manual/21-output-settings.png)

3. **生成与结果**  
   - 点击生成后等待完成；失败时阅读错误提示（如积分不足、网络超时）。  
   - 成功后通常可 **保存到本地**，且图片会 **进入仓库**。

4. **从仓库带图进入**  
   - 在仓库对某张图使用 **用AI编辑**，在菜单中选择功能，即会打开对应 AI 美工页并自动加载该图。

### 4.2 图片编辑

| 功能 | 操作要点 |
|------|----------|
| 局部重绘 | 侧栏 **局部重绘**：上传原图，涂抹 **遮罩**，填写修改说明，生成替换遮罩区域。 |
| 局部消除 | 侧栏 **局部消除**：上传原图，涂抹需去除区域，生成抹除后效果。 |
| 一键换色 | 侧栏 **一键换色**：上传产品图，按提示选择或描述目标颜色。 |
| 服装 3D | 侧栏 **服装 3D**：上传服装图，按页面说明生成立体展示效果。 |
| 服装平铺 | 侧栏 **服装平铺**：上传服装图，生成平铺展示图。 |
| 调整身材 | 侧栏 **调整身材**：上传模特穿搭图，按滑块或选项调整体型展示。 |
| 生成场景 | 侧栏 **生成场景**：上传产品图，填写 **多个场景描述**（有上限），可批量生成多场景图。 |
| 添加人/物 | 侧栏 **添加人/物**：上传产品图，配置人物/物品数量、位置与描述（或参考图），生成合成图。 |

### 4.3 质量提升

| 功能 | 操作要点 |
|------|----------|
| 智能扩图 | 侧栏 **智能扩图**：上传图，指定扩展方向或比例，补全画面。 |
| 提升质感 | 侧栏 **提升质感**：上传产品图，选择材质预设或自定义描述，增强质感。 |

### 4.4 文字修改

| 功能 | 操作要点 |
|------|----------|
| 文字替换 | 侧栏 **文字替换**：指定原文字与目标文字（或按页面表单项填写）。 |
| 语言转换 | 侧栏 **语言转换**：将图上文字译为其他语言；页面可能对模型有推荐（如优先 Nano Banana 2）。 |
| 文字去除 | 侧栏 **文字去除**：去除图中文字；复杂背景建议选更高能力模型。 |

### 4.5 风格变迁

| 功能 | 操作要点 |
|------|----------|
| 风格复刻 | 侧栏 **风格复刻**：上传 **参考风格图** + **产品图**，分阶段生成与参考风格一致的产品图。 |
| 风格改变 | 侧栏 **风格改变**：上传图，描述目标风格或选择预设。 |

### 4.6 水印

| 功能 | 操作要点 |
|------|----------|
| 添加水印 | 侧栏 **添加水印**：上传图，设置水印文字或样式（以页面为准）。 |
| 去除水印 | 侧栏 **去除水印**（或官方示例中的去水印流程）：按页面指引上传图并生成。 |

### 4.7 官方示例（修改图片 · 10 种模式）

以下入口在侧栏 **官方示例** 中，页顶一般有 **Before / After** 对照示例，按名称进入即可。

| 模式 | 说明摘要 |
|------|----------|
| 添加/移除元素 | 自然语言描述增删物体。 |
| 局部重绘(语义) | 偏语义理解的局部修改。 |
| 风格迁移 | 整图风格变换。 |
| 高级合成 | 多图合成一张。 |
| 高保真细节保留 | 在保留主体细节下融合元素。 |
| 让草图变生动 | 草图转成品感图像。 |
| 角色一致性 360° | 基于参考图生成不同视角。 |
| 文字替换 | 同 4.4。 |
| 语言转换 | 同 4.4。 |
| 去除水印 | 去除明显水印。 |

![图：修改图片模式页示例](images/user-manual/40-imageedit-demo.png)

**操作步骤（通用）**：

1. 选择模式，阅读页顶示例。  
2. 按提示上传 **一张或多张** 图。  
3. 填写 **英文或中文提示**（以页面占位符为准）；需要遮罩的模式完成涂抹。  
4. 设置 **输出设置**，点击生成。  
5. 保存结果并到仓库查看。

## 五、电商 AI 运营助手

**路径**：顶栏 **电商AI运营助手**。

进入后，在 **左侧选择平台**：**亚马逊**、**eBay**、**速卖通**。不同平台下列出的功能卡片不同。

![图：平台切换与功能入口](images/user-manual/30-ai-assistant-platforms.png)

### 5.1 亚马逊

![图：亚马逊功能卡片](images/user-manual/31-amazon-generate-cards.png)

#### 5.1.1 生成 Listing

典型步骤（以页面为准）：

1. **步骤 1 · 分析**  
   - 上传至少一张产品图，填写类目、品牌、卖点、市场、语言等表单。  
   - 提交分析，等待返回产品理解与结构化信息。

2. **步骤 2 · 标题 / 关键词 / 五点 / 描述**  
   - 在分析完成后，生成或编辑标题、后台关键词、五点、长描述。  
   - 文案多为 **纯文本**，便于粘贴到卖家后台。

3. **步骤 3 · 产品图**  
   - 按页面配置主图与各类型张数（与通用生图逻辑类似处，以当前页为准）。  
   - 生成会扣积分；成功后图一般进 **仓库**。

4. **步骤 4 · A+ 文案与图片**（若您的账号已开放本步，页面会出现；否则可跳过）  
   - 选择亚马逊标准 A+ 模块（最多 5 个），生成文案与配图；消耗积分以页面为准。  
   - 若顶栏有 **A+ 页面** 入口，也可从该入口独立走完整 A+ 流程。

5. **保存与导出**  
   - **保存到我的 Listing**：将本套文案与关联图片 ID 写入 **Listing 历史**。  
   - **导出 CSV / JSON**：导出文案字段（图片请在仓库下载）。

#### 5.1.2 优化 Listing

1. 粘贴或填写已有标题、五点、描述等。  
2. 可使用 **智能粘贴**（若提供）：从平台后台复制整页，由 AI 提取字段。  
3. 运行优化，查看诊断报告与优化建议；该能力一般 **不扣积分**（以页面说明为准）。  
4. 在结果区可查找 **生成 A/B 变体** 等入口，对比多套文案。

#### 5.1.3 竞品对比

1. 填写自己的 Listing 与 1～5 个竞品内容（支持 Tab 切换输入）。  
2. 生成对比报告：关键词差异、卖点矩阵、策略建议等。  
3. 一般不扣积分。

#### 5.1.4 关键词研究

1. 输入产品名称或核心词。  
2. 查看分组关键词、长尾建议、标题排布思路等。  
3. 一般不扣积分。

### 5.2 eBay

- **生成 Listing**：上传图片、填写表单 → 分析与生成 → 保存 / 导出。  
- **优化 Listing**：粘贴现有文案 → 诊断与优化建议；支持与亚马逊类似的 **智能粘贴** 体验（以页面为准）。

### 5.3 速卖通

- **生成 Listing**：流程与 eBay 类似，字段符合速卖通习惯（标题长度、属性等）。  
- **优化 Listing**：针对速卖通搜索与合规的诊断与建议。

### 5.4 Listing 历史

在工作台 **Listing 历史** 中可查看已保存记录、复制文案、查看关联主图与扩展配图（若有）。详见 [第六节](#六工作台仓库listing-历史积分明细)。

![图：Listing 历史](images/user-manual/32-listing-history.png)

## 六、工作台：仓库、Listing 历史、积分明细

**进入方式**：顶栏 **工作台**（进入后默认打开仓库）。

![图：工作台侧栏](images/user-manual/04-dashboard-sidebar.png)

### 6.1 仓库

1. 图片按 **保存日期** 分组展示。  
2. **全选**：选中当前列表全部图片。  
3. **选中当日全部**：某一日期标题旁可勾选，仅选中该日全部图片。  
4. 也可单张勾选。  
5. **保存选中到本地**：批量下载（浏览器可能要求选择文件夹）。  
6. **删除选中**：批量删除服务器上的记录与文件（谨慎操作）。  
7. 单张图：**用AI编辑**（弹出与 AI 美工一致的功能菜单）、**下载**、**移除**。  
8. 点击缩略图可 **放大预览**；若加载失败可 **重试**。

![图：仓库列表与按日选择](images/user-manual/22-gallery.png)

![图：用AI编辑菜单](images/user-manual/23-gallery-ai-edit-menu.png)

### 6.2 Listing 历史

1. 浏览已保存的 Listing 记录列表。  
2. 进入详情查看标题、五点、描述、关联图片等。  
3. 使用 **复制** 按钮将各字段复制到剪贴板，粘贴到平台后台。

### 6.3 积分明细

1. 查看积分 **收入 / 支出** 流水与余额相关说明。  
2. 对扣费有疑问时，可结合时间与描述核对。

## 八、浏览器扩展（可选）

适用于 **Chrome / Edge** 等 Chromium 浏览器（Manifest V3）。

### 8.1 安装与站点地址

1. 登录官网，在顶栏 **浏览器扩展** 下载 zip（无需向运营索取附件）；解压后于 Chrome/Edge「扩展程序」中开启开发者模式，选择 **加载已解压的扩展程序**。后续若上架应用店，可按官方说明安装。  
2. 若扩展要求填写网站地址，请使用 **https://pictoolai.studio**（与当前登录站点一致）；开发者本地调试另见 [BROWSER-EXTENSION.md](./BROWSER-EXTENSION.md)。

### 8.2 使用步骤

1. 在任意网页对 **图片** 右键。  
2. 选择 **PicToolAI** 子菜单：  
   - 可直接进入某一 **AI 美工** 功能；或  
   - **通用电商生图** 下选择 **白底主图 / 特写图 / 卖点图 / 场景图 / 交互图**，将带图打开组图页并预选类型。  
3. 浏览器跳转至 PicToolAI；已登录则自动传图；若失败，请确认网站地址配置与登录状态。

![图：右键 PicToolAI 菜单](images/user-manual/50-extension-menu.png)

更完整的扩展说明、权限与调试见 [BROWSER-EXTENSION.md](./BROWSER-EXTENSION.md)。

## 九、积分与模型简要说明

- **积分**：生图、部分深度检测等会扣积分；**Listing 文案生成/优化、竞品对比、关键词研究** 等多数分析能力常以 **不扣积分** 方式提供（以官网 **定价页** 与各页提示为准）。  
- **模型**：常见为 **Nano Banana / Nano Banana 2 / Nano Banana Pro**；不同模型支持的 **清晰度挡位** 不同，积分单价也不同。  
- **套餐与单价**：对外介绍与积分表见 [PRODUCT-INTRO.md](./PRODUCT-INTRO.md)；站内定价以 **订阅页** 为准。
