PicToolAI 模型选择指南：质感增强篇

提升质感功能支持 Nano Banana Pro 与 Nano Banana 2 两种模型。以下内容帮助你根据场景做出选择。

一、专家建议：黄金法则

不确定如何选择时，可按以下三类场景对应：

处理对象 建议模型 说明
精密零件、珠宝、高级家具 Nano Banana Pro 能捕捉微小的加工痕迹和复杂的环境反射，让物体看起来「很贵」。
美食、风景、时尚人像 Nano Banana 2 色彩处理更讨喜，光影通透，能让画面看起来非常清新、高级。
只想让模糊的图变清晰 先试 Nano Banana 2 边缘锐化处理非常干练，且不需要等待太久。

二、详细对比：Pro vs 2

维度 Nano Banana 2（速度优先型） Nano Banana Pro（极致细节型）
底层引擎 Gemini 3.1 Flash Image Gemini 3 Pro Image
核心优势 快如闪电、极高效率。在保持约 95% 质感的前提下，生图速度约快 3～5 倍 深层推理、材质大师。能理解金属折射、织物纤维等复杂物理属性
质感表现 现代感强，光影明亮生动，色彩饱和度高，适合快速出图 极度真实，纹理细节（如划痕、锈迹、颗粒）更加深邃且富有层次
推荐场景 批量处理、社交媒体分享、初步预览；追求「一眼惊艳」的视觉效果 商业摄影、工业设计、微距细节、4K 画质需求；追求「经得起放大的真实感」
生成时间 约 4～6 秒（1K） 约 10～20 秒（含深度运算）
指令遵循 对简洁指令反应极快 擅长处理超长、复杂的材质描述指令

三、快速决策

  • 追求质感与真实感 → 选 Pro
  • 追求速度与效率 → 选 2
  • 金属、珠宝、精密件 → 选 Pro
  • 美食、风景、人像 → 选 2

四、用户材质提示词（可选）

提升质感支持用户填写或选择材质关键词，采用「通用质感包 + 用户材质」的权重融合：

  • 前端：材质预设标签（工业 / 自然 / 轻奢 / 塑料 四个 Tab，共 23 项）+ 自定义输入框
  • 后端：materialPrompt 非空时，追加 The subject is made of {materialPrompt}.；若检测到塑料相关词汇，则额外强制追加 SSS、Soft-touch、Beveled edges 三个塑料质感增强词。

用户不填时使用纯通用 prompt；填写后可避免材质冲突（如塑料件被误判为钢铁）、支持材质转换（如木头→金属）。

详细说明（预设列表、塑料专项 Tab、塑料黑科技增强逻辑）见 AI-DESIGNER-TEXTURE-MATERIALS.md。
